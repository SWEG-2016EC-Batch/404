## Pseudocode for full pyramid

Start

### OuterLoop
Initialize i = 1

If i > 6 then go to EndProgram

Initialize j = 1

### InnerLoop
If j > (7 - i) then go to NewLine

Output "* "

Increment j by 1

Go to InnerLoop

### NewLine
Output new line

Increment i by 1

Go to OuterLoop

### EndProgram
End Program
