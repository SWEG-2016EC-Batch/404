## Pseudocode for number right triangle

Start

Initialize i = 1
### Input
    Read rows
    
### OuterLoop
If i > rows then go to EndProgram

Initialize j = 1

### InnerLoop
If j > i then go to NewLine

Output j

Increment j by 1

Go to InnerLoop

### NewLine
Output new line

Increment i by 1

Go to OuterLoop

### EndProgram
End Program
